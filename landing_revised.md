## 1. 서비스 개요

### 1.1 서비스명

**심리상담연구소 앤아더라이프** (도포 x 심리상담연구소 앤아더라이프 콜라보)

### 1.2 서비스 컨셉

4만 팔로워를 보유한 캐릭터 IP '도포'와 협업하여, 2030 세대를 위한 합리적인 가격의 전문 심리상담 서비스를 제공하는 랜딩 웹사이트

### 1.3 핵심 가치 제안

- **첫 상담 무료** (상담 적합성 확인 및 라포 형성)

- **전문가 슈퍼비전** (교수급 지도 감독 필수)
- **비대면 · 대면 상담** 모두 지원

## 5. 상담 신청 모달 (폼)

### 5.1 개요

- **형태**: 3단계 스텝 폼
- **프로그레스바**: 상단에 3단계 진행 상황 표시 (현재 단계 강조)

### 5.2 Step 1: 기본 정보

**안내 문구**: "상담 연락을 위해 정보를 입력해주세요."

| 필드     | 타입        | 필수 | 플레이스홀더    | 유효성 검사                      |
| -------- | ----------- | ---- | --------------- | -------------------------------- |
| 이름     | 텍스트      | O    | "홍길동"        | 2글자 이상                       |
| 연락처   | 텍스트      | O    | "010-1234-5678" | 휴대폰 번호 형식 (01X-XXXX-XXXX) |
| 출생연도 | 숫자        | O    | "1995"          | 4자리 숫자                       |
| 성별     | 라디오 버튼 | O    | -               | 남성 / 여성 / 기타 중 선택       |

### 5.3 Step 2: 상담 방식 및 일정

**안내 문구**: "상담은 매주 같은 요일/시간에 진행됩니다."

| 필드          | 타입          | 필수 | 옵션                                                                                           |
| ------------- | ------------- | ---- | ---------------------------------------------------------------------------------------------- |
| 상담 방식     | 라디오 버튼   | O    | 대면 / 비대면 / 상관없음                                                                       |
| 선호 지역     | 텍스트        | X    | 대면 또는 상관없음 선택 시만 표시<br>플레이스홀더: "예: 홍대, 강남 등"                         |
| 가능한 요일   | 복수선택 버튼 | O    | 월 / 화 / 수 / 목 / 금 / 토                                                                    |
| 가능한 시간대 | 복수선택 버튼 | O    | - 오전 (10시~12시)<br>- 오후 초반 (12시~15시)<br>- 오후 후반 (15시~17시)<br>- 저녁 (17시~21시) |

### 5.4 Step 3: 상담 주제

**안내 문구**: "어떤 고민으로 상담을 원하시나요?"

| 필드      | 타입              | 필수 | 옵션                                                                                                                                                                                                         |
| --------- | ----------------- | ---- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| 고민 선택 | 복수선택 체크박스 | O    | - 진로/직장 (취업, 이직, 직장 내 스트레스)<br>- 대인관계 (가족, 연인, 친구와의 갈등)<br>- 우울/불안 (우울감, 불안, 무기력)<br>- 자기이해 (자존감, 정체성)<br>- 기타 (일상에서 설명하기 어려운 심리적 불편감) |
| 부가 설명 | 텍스트에어리어    | X    | 플레이스홀더: "상담사에게 미리 전달하고 싶은 내용이 있다면 적어주세요."                                                                                                                                      |

**사전 확인 질문**:
